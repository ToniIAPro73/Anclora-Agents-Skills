# Habilidad: Creador de Blogs

Esta habilidad transforma ideas sencillas o temas complejos en artículos de blog profesionales y atractivos. Combina redacción persuasiva, optimización para motores de búsqueda (SEO) y diseño visual dinámico.

## Capacidades

- **Planificación de Contenido**: Creación de esquemas lógicos y fluidos (Storytelling).
- **SEO On-Page**: Selección de palabras clave, redacción de meta-descripciones y estructura de encabezados.
- **Visual Storytelling**: Identificación de momentos clave para integrar imágenes y creación de prompts artísticos.
- **Adaptación de Tono**: Capacidad para escribir en tonos educativos, corporativos, técnicos o inspiracionales.

## Instrucciones de Uso

1.  **Definición**: Recibe el tema, palabra clave o idea principal del usuario.
2.  **Estructuración**: Crea un esquema usando la plantilla `resources/plantilla_blog.md`.
3.  **Redacción**: Desarrolla el contenido asegurando una introducción potente y secciones bien conectadas.
4.  **Activos Visuales**: Diseña prompts específicos para la cabecera (featured image) y al menos un visual intermedio.
5.  **Generación**: Utiliza `generate_image` para crear estas piezas visuales.
6.  **Revisión Final**: Asegura que el formato (negritas, listas, enlaces) mejore la legibilidad.

## Mejores Prácticas

- **Encabezados**: Usa un solo H1 y múltiples H2/H3 para jerarquizar.
- **Párrafos Cortos**: No exceder las 3-4 líneas para facilitar la lectura digital.
- **Alt Text**: Sugiere textos alternativos para las imágenes generadas.
- **CTA**: Incluye siempre una llamada a la acción al final del post.
